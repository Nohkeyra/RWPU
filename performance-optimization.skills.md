# Performance Optimization Skill

## Objective
Optimize only after identifying measurable bottlenecks.

## Workflow
1. Reproduce the issue.
2. Measure (CPU, memory, network, rendering, bundle size, DB queries).
3. Identify the root cause.
4. Propose the smallest effective optimization.
5. Benchmark before vs. after.
6. Verify no regressions.

## Checklist
- Avoid premature optimization.
- Remove unnecessary work before adding complexity.
- Consider caching, lazy loading, batching, indexing, memoization, and pagination where appropriate.
- Document trade-offs.

## Expected Output
- Root cause
- Evidence
- Optimization plan
- Estimated impact
- Validation results
